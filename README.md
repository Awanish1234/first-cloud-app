# ATSS COLLEGE Chatbot

This is a Streamlit-based college chatbot project with student and admin portals.

## Features
- Role-based login (Student / Admin)
- Secure password hashing with bcrypt
- Oracle database integration
- Admin panel with chat history and filters
- Student chatbot with department-specific responses

## Run
1. Install dependencies: `pip install -r requirements.txt`
2. Set Oracle DB credentials in `.env`
3. Run: `streamlit run atss_chatbot.py`
